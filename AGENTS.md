# ai-tg-channels

## После каждой задачи

Обязательный чеклист перед коммитом:

1. **Typecheck** — `npm run typecheck`
2. **Lint** — `npm run lint`
3. **Тесты** — `npm test`; если добавлена нетривиальная логика — дописать тест в `tests/`
4. **Коммит** — только если все три проверки выше прошли без ошибок

Если какая-то проверка упала — исправить до коммита.
